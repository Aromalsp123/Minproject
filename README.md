# MicroFormerX вҖ” Microplastic Detection & Classification

> AI-powered detection and classification of microplastics in microscopy images using a dual-backbone deep learning architecture.

---

## Overview

**MicroFormerX** is an end-to-end deep learning pipeline designed to detect and classify microplastic particles from microscopy images вҖ” particularly those containing algae. The system combines a **Swin Transformer** and **ConvNeXt** backbone in a fused architecture to classify microplastics into four categories, while also providing explainable AI (XAI) visualizations and microplastic detection inside algae cells.

## Features

- **Dual-Backbone Architecture**: Swin Transformer + ConvNeXt feature fusion
- **4-Class Classification**: `algae`, `filament`, `fragment`, `pellet`
- **Multi-Task Learning**: Morphology classification + source detection heads
- **Explainable AI**: GradCAM (EigenCAM) and LIME for model interpretability
- **Algae Segmentation**: Automatic cell segmentation with microplastic particle detection
- **Morphology Analysis**: Circularity, aspect ratio, solidity, area extraction
- **Flask Web App**: Drag-and-drop upload, camera capture, real-time analysis with annotated overlays
- **External Validation**: Robustness testing with synthetic realistic variations

## Model Architecture

**MicroFormerX** uses a two-stream feature extraction pipeline:

```
Input Image (224Г—224)
    в”ңв”Җв”Җ Swin-Tiny Backbone  вҶ’  feature vector
    в””в”Җв”Җ ConvNeXt Tiny       вҶ’  feature vector
                 вҶ“
          Concatenation
                 вҶ“
         Fusion MLP (512-d)
                 вҶ“
    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
    вҶ“                         вҶ“
Morphology Head          Source Head
(4 classes)              (3 sources)
```

- **Backbones**: Pretrained `swin_tiny_patch4_window7_224` + `convnext_tiny` (via `timm`)
- **Fusion**: Concatenated features вҶ’ Linear(512) вҶ’ ReLU вҶ’ Dropout(0.3)
- **Heads**:
  - `morph_head`: Classifies microplastic type (4 classes)
  - `source_head`: Detects image source domain (3 sources)

## Usage

### 1. Training

```bash
python train.py
```

- Saves best model to `outputs/best_model.pth`
- Saves final epoch model to `outputs/microformerx_rtx3050.pth`
- Logs training metrics to `outputs/training_log.csv`

**Training configuration** (from `config.py`):

- Epochs: 20
- Batch size: 4
- Learning rate: 1e-4
- Optimizer: AdamW with cosine annealing
- Augmentation: Random crop, horizontal/vertical flip, color jitter
- AMP (Automatic Mixed Precision) enabled on CUDA

### 2. Inference

```bash
python infer.py
```

Predicts the class of a single image. Modify the `image_path` in the script or import `predict()` in your own code.

### 3. External Evaluation

```bash
python evaluate.py
```

Evaluates the model on the `external_test` set with harder transformations (brightness/contrast jitter, Gaussian blur) and generates a confusion matrix (`external_confusion_matrix.png`).

### 4. Create External Test Set

```bash
python create_external_dataset.py
```

Generates a robust external validation set by applying realistic variations:

- Random brightness & contrast scaling
- Gaussian blur
- Mild Gaussian noise

Samples 20 images per class from the validation set by default.

### 5. Explainable AI (XAI) Analysis

```bash
python xai.py
```

Generates a comprehensive 2Г—3 panel visualization saved to `outputs/xai_result.png`:

- Original image
- GradCAM heatmap (model attention)
- LIME explanation
- Algae cell segmentation
- Detected microplastics inside algae cells
- Summary statistics panel

Also performs:

- Algae cell segmentation with CLAHE + Otsu + morphological cleanup
- Microplastic particle detection & classification by shape
- Morphological statistics (circularity, aspect ratio, solidity, area)

### 6. Visual Analysis

```bash
python visual_analysis.py
```

Creates a 3-panel visualization:

- Original image
- Detection overlay (contour + bounding box + label)
- Detailed analysis overlay with morphology stats

Saved to `outputs/visual_analysis.jpg`.

### 7. Web Application

```bash
python app.py
```

Launches the Flask backend on `http://localhost:5000`.

**Features:**

- Drag-and-drop image upload
- Camera capture support
- Real-time classification with confidence scores
- Morphology analysis panel
- Annotated detection image with contour & bounding box
- Analysis history (stored in browser localStorage)
- Responsive premium dark UI

**API Endpoints:**

| Method | Endpoint       | Description                                  |
| ------ | -------------- | -------------------------------------------- |
| GET    | `/`            | Frontend HTML                                |
| GET    | `/api/status`  | Model health & device info                   |
| GET    | `/api/classes` | List of class names                          |
| POST   | `/api/predict` | Upload image вҶ’ JSON prediction               |
| POST   | `/api/analyze` | Upload image вҶ’ JSON + annotated base64 image |

## Microplastic Detection Inside Algae

The `xai.py` pipeline includes a specialized detector that:

1. **Segments algae cells** using CLAHE enhancement, Otsu thresholding, and morphological operations
2. **Detects microplastics within each cell** using multi-signal thresholding (adaptive, local std-dev, Otsu)
3. **Classifies each particle** by morphology:
   - **Pellet**: Circularity > 0.70, Aspect Ratio < 2.0
   - **Filament**: Aspect Ratio > 3.5 or Elongation > 0.68
   - **Fragment**: Solidity < 0.78
   - **Fiber/Other**: Remaining cases
4. **Generates annotated visualizations** with color-coded bounding boxes, centroids, and labels
